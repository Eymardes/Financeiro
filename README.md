# Financeiro

App simples de organização financeira pessoal.

## Funcionalidades

- Receitas
- Despesas
- Parcelamentos
- Assinaturas recorrentes
- Dívidas
- Histórico
- Resumo mensal e anual
- Calendário

## Como usar

1. Baixe o projeto
2. Abra o arquivo `financeiro.html`
3. Use normalmente no navegador

## Salvamento

Os dados ficam salvos no navegador (`localStorage`).

Também é possível:
- baixar backup em JSON
- carregar backup
- salvar em pasta local

## Estrutura

financeiro.html

data/

## Tecnologias

HTML + CSS + JavaScript puro

## Limitações
Não precisa instalar nada
Não usa servidor
Funciona melhor no Chrome e Edger.
