Pasta de dados

Esta pasta guarda o banco de dados do app. Não apague o arquivo database.json 
Para conectar esta pasta ao app:
1. Abra financeiro.html no Chrome ou Edge
2. Role até o final da página e clique em "Conectar pasta"
3. Selecione esta pasta (data)
4. Clique em "Permitir" quando o navegador pedir acesso
