Pasta de dados do app Financeiro.

Esta pasta guarda o banco de dados do app. Não apague o arquivo database.json — ele contém todos os seus lançamentos, parcelas, dívidas e histórico.

Arquivos:
- database.json     -> banco de dados principal (atualizado pelo botão "Salvar dados")
- backup-*.json     -> snapshots datados, criados também pelo botão "Salvar dados"

Para conectar esta pasta ao app:
1. Abra financeiro.html no Chrome ou Edge
2. Role até o final da página e clique em "Conectar pasta"
3. Selecione esta pasta (data)
4. Clique em "Permitir" quando o navegador pedir acesso
